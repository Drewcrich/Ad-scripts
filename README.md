## ğŸ¬ Script Creation Rules (Critical)

1. **Clip durations: ONLY 10s or 15s** (Sora 2 limitation)
2. **Always include background music** in every prompt
3. **Dialogue limits:** 25-30 words (10s), 35-45 words (15s)
4. **Keep prompts SIMPLE** - Let Sora fill in details naturally
5. **Fish on deck only** - Never show fish in water
6. **AI transparency** - Always include disclosure
7. **Content-first** - Hook should feel like fishing content, not an ad

**See `AI_GUIDE.md` for complete rules and examples.**
